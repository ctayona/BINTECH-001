
BinTECH User Stories (Polished and Expanded)




Student / Faculty / User


User Type	Epic/Feature	User Story	Priority	Complexity	Estimate/Story Points	Status
Student/Faculty/User	Account Registration (QR + Email)	As a user, I can register an account after scanning a registration QR code so onboarding is quick and kiosk-ready.	Must	Medium	4	100%
Student/Faculty/User	Account Registration (Profile Data)	As a user, I can input and save my basic profile details (name, email, role-linked identity fields) to complete registration.	Must	Low	3	100%
Student/Faculty/User	Authentication & Session	As a user, I can securely log in and log out so my account data and points are protected.	Must	Low	3	100%
Student/Faculty/User	Password Recovery	As a user, I can request OTP-based password recovery and reset my password if I forget it.	Must	Medium	4	100%
Student/Faculty/User	Device Pairing & Verification	As a user, I can pair to a specific kiosk by scanning its device QR so my session links to the correct machine.	Must	Low	3	100%
Student/Faculty/User	Kiosk Session Start	As a user, I can scan the machine QR code to connect to a sorting session and start tracking disposal activity.	Should	High	5	100%
Student/Faculty/User	Waste Disposal	As a user, I can insert waste into the kiosk for automatic material detection and sorting.	Must	Medium	4	100%
Student/Faculty/User	Sorting Feedback	As a user, I can receive immediate feedback when my waste is correctly sorted.	Must	Medium	4	100%
Student/Faculty/User	Live Session Counters	As a user, I can see live session counters (metal, plastic, paper, session points) while sorting.	Should	Medium	4	100%
Student/Faculty/User	Rewards Accrual	As a user, I can earn points after properly disposing of waste.	Must	Medium	4	100%
Student/Faculty/User	Points Visibility	As a user, I can view my current and accumulated reward points from dashboard and navigation widgets.	Must	Low	3	100%
Student/Faculty/User	Reward Redemption	As a user, I can redeem points for rewards (e.g., snacks, vouchers) and receive coupon details.	Must	Medium	4	100%
Student/Faculty/User	Email Coupon Delivery	As a user, I can receive reward coupon information via email after redemption.	Should	Medium	4	100%
Student/Faculty/User	Peer Transfer	As a user, I can transfer points to another student using recipient email validation.	Could	Low	3	100%
Student/Faculty/User	Safe Transfer / Retry	As a user, I can cancel or retry point transfers if disconnection happens mid-transfer so points remain consistent.	Must	Medium	4	100%
Student/Faculty/User	Session Resume	As a user, I can resume my active kiosk session after temporary network loss so pending points are preserved.	Must	Medium	4	100%
Student/Faculty/User	Offline Buffering & Reconciliation	As a user, my kiosk can buffer sorting events while offline and auto-sync when online so no earned points are lost.	Must	High	5	100%
Student/Faculty/User	Profile Management	As a user, I can edit my profile information, upload profile picture, and save changes.	Must	Medium	4	100%
Student/Faculty/User	Activity Statistics	As a user, I can view my activity statistics from the user dashboard.	Should	Medium	4	100%
Student/Faculty/User	Transaction History	As a user, I can view transaction history for reward redemptions, machine sessions, and peer transfers with summary totals.	Must	Low	3	100%
Student/Faculty/User	Leaderboard	As a user, I can view leaderboard standings and my current rank to track performance.	Should	Medium	4	100%
