## Data Calculations

### Percentages
```
Percentage = (Value / Total) × 100
Format: 2 decimal places (e.g., 45.67%)
```

### Proportions
```
Proportion = Value / Total
Format: 4 decimal places (e.g., 0.4567)
Verification: Sum of all proportions = 1.0000
```

### Weekly Averages
```
Average per Material = Weekly Total / 3
Format: 2 decimal places
```

### Trend Analysis
```
Average Weekly Waste = Total Waste / Number of Weeks
Peak Week = Week with highest total
Lowest Week = Week with lowest total
```

## Usage

1. **Navigate to Analytics Page**: `/admin/analytics`
2. **Select Date Range**: Use date pickers for from/to dates

4. **Choose Export Format**:
