## Export File Names

All exports use consistent naming:
```
BinTECH_Analytics_[dateFrom]_to_[dateTo].[format]
```

Examples:
- `BinTECH_Analytics_2026-05-01_to_2026-05-31.csv`
- `BinTECH_Analytics_2026-05-01_to_2026-05-31.pdf`
- `BinTECH_Analytics_2026-05-01_to_2026-05-31.xlsx`

---

## Data Calculations

All percentages are calculated as:
```
(Value / Total) * 100
```

Formatted to 2 decimal places for precision.
