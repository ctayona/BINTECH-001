## Interaction Flow

### Editing Student Account with COR

```
1. Click "Edit" on student account
   ↓
2. Edit modal opens
   ↓
3. Modal populated with student data
   ↓
4. COR URL loaded
   ↓
5. COR Full Preview section appears
   ↓
6. User sees thumbnail + "View Full" link
   ↓
7. User clicks thumbnail
   ↓
8. Full-screen preview modal opens
   ↓
9. User closes preview (X, Escape, or background click)
   ↓
10. Back to edit modal
   ↓
11. User makes changes
   ↓
12. User clicks "Save Changes"
   ↓
13. Account updated
```

## Keyboard Navigation

| Key | Action |
|-----|--------|

| Enter | Submit form / Open dropdown |
| Escape | Close modal / Close preview |
